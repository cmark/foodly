# Foodly Sample Single Page Web Application

<a name="installation"/>
## Usage

Clone github repository,

```
$ git clone git@github.com:CMark/foodly.git
```

Install npm dependencies,

```
$ npm install
```

Install bower dependencies,

```
$ bower install
```

Run app (development mode),

```
$ node app.js
```
